# Lesson 24. Interfaces

## Task 1

Write a program that outputs the coordinates (from previous chapters) in JSON format, e.g.

``` json
{
    "decimal": 135.9,
    "dms": "135o54'0.0\" E",
    "degrees": 135,
    "minutes": 54,
    "seconds": 0,
    "hemisphere": "E"
}
```
